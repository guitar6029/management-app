To set up front end :

```
npm install
```


to set up back end :

first create a virtual env in the backend root directory : 

```
python -m venv venv

```

activate the env : 

windows - 

```
venv\Scripts\activate

```

bash - 

```
source venv/bin/activate

```

install the packages :

```
pip install -r requirements.txt

```
